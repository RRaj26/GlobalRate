# 🌍 GlobalRate — Real-Time Currency Dashboard

A premium, interactive, real-time currency converter and exchange rate tracking dashboard. Built with React 19, Tailwind CSS v4, Framer Motion, and Recharts.

🔗 **Live Demo:** [https://global-rate.vercel.app/](https://global-rate.vercel.app/)

---

## ✨ Features

- **Real-Time Exchange Rates:** Fetches up-to-date exchange rates for over 150+ fiat currencies globally.
- **Interactive Conversions:** Seamless two-way input conversion with instant swapping of currencies.
- **Beautiful Trend Visualizations:** Interactive 7-day historical rate trends rendered dynamically via Recharts area charts.
- **Favorite & Popular Currencies:** Quick-access widget for major currencies (USD, EUR, GBP, INR, etc.) directly in the dashboard.
- **Premium User Interface:** Dark-mode first design system with responsive grid layouts, glassmorphic card effects, and elegant micro-animations powered by Framer Motion.
- **Flag Integration:** Automatically matches national currency codes to their respective emojis/flags.

---

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vite.dev/)
- **Styling:** [Tailwind CSS v4.0](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Charts:** [Recharts](https://recharts.org/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **API Data Source:** [Fawaz Ahmed's Currency API](https://github.com/fawazahmed0/currency-api)

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/RRaj26/GlobalRate.git
cd GlobalRate
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```
